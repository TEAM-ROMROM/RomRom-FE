# ®️ROMROM®️

<br/>

## 🙂 프로젝트 소개
롬롬은 현금, 물건, 재능, 서비스를 모두 교환할 수 있는 혁신적인 플랫폼입니다. 

사용자들은 스와이프 탐색과 AI 기반 가격 제안, 드래그 앤 드롭 요청 기능을 통해 빠르고 간편하게 원하는 교환을 실현할 수 있습니다. 

ROMROM은 물건 교환을 넘어선, 재능과 서비스까지 포괄하는 새로운 교환 문화를 제시합니다.
