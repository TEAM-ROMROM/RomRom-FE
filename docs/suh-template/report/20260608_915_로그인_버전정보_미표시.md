# 로그인 화면 우하단 버전 정보 미표시

## 개요
로그인 화면(스플래시 후 노출되는 로그인 UI)의 우하단에 앱 버전 정보가 표시되지 않던 문제를 해결했다. `LoginScreen`과 동일한 방식으로 `PackageInfo`에서 버전을 읽어 우하단에 고정 픽셀 위치로 표시한다.

## 변경 사항

### 버전 로드
- `lib/screens/splash_screen.dart`: `package_info_plus`로 `_loadAppVersion()` 추가. `initState`에서 비동기 로드하며, 실패 시 무시(빈 문자열 유지)하고 `mounted` 가드 적용.

### UI
- `lib/screens/splash_screen.dart`: 로그인 UI를 `Stack`으로 감싸 기존 `Column`(로고·문구·버튼) 위에 우하단 버전 텍스트를 오버레이. `Positioned(right: 20, bottom: 40)` 고정 픽셀, `p4` 스타일·흰색 40% 투명도. `_appVersion.isNotEmpty`일 때만 렌더.

## 주요 구현 내용
- 버전 표시는 `LoginScreen`과 동일 구현으로 일관성 유지.
- iPad 대응을 위해 `.w`/`.h` 대신 **고정 픽셀**(`right: 20`, `bottom: 40`)을 사용해 대형 기기에서 위치가 과도하게 커지지 않도록 했다.
- 버전 로드 실패 시 크래시 없이 텍스트만 미표시되도록 try/catch + 빈 문자열 가드.

## 주의사항
- `PackageInfo` 로드는 비동기라 화면 진입 직후 짧은 순간 버전이 비어 있을 수 있으나, 로드 완료 시 `setState`로 즉시 반영된다.
