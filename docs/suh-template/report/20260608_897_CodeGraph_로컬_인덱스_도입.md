# CodeGraph 로컬 인덱스 도입 및 팀 사용 가이드

## 개요
AI 코딩 에이전트(Claude Code, Cursor, Codex 등)가 `grep`/`Read` 대신 사전 인덱싱된 코드 그래프를 직접 쿼리하도록 하는 CodeGraph를 프로젝트에 도입하고 팀 사용 가이드를 정리했다. 문서/DX 개선 작업으로 앱 동작 변경은 없다.

## 도입 효과
- 평균 25% 비용 절감, 62% 적은 툴 콜
- 100% 로컬 (외부 전송 없음, API 키 불필요)
- 20+ 언어 지원 (Dart/Flutter Full Support)

## 변경 사항

### 저장소 설정
- `.gitignore`: `.codegraph/` 추가(커밋 `2d1dbd7`)로 로컬 인덱스 DB(`.codegraph/codegraph.db`)가 커밋되지 않도록 처리.

### 가이드 (이슈 본문)
- 글로벌 설치(`npm i -g @colbymchenry/codegraph`) → Claude Code MCP 등록(`codegraph install --target=claude --location=global`) → 재시작 → 프로젝트 인덱싱(`codegraph init -i`) 절차 문서화.
- 주요 MCP 툴(`codegraph_search`/`context`/`trace`/`callers`/`callees`/`impact`/`explore`)·CLI 명령어 정리.

## 주요 구현 내용
- 파일 변경 시 native OS 이벤트(FSEvents/inotify)로 자동 감지 후 2초 debounce 증분 sync — 수동 동기화 불필요.
- PC당 1회 글로벌 설치 + MCP 등록, 프로젝트당 1회 인덱싱이면 팀원 모두 적용 가능.

## 검증
- `codegraph init -i` 실행으로 인덱싱 성공 확인 (예: 626 nodes, 1,428 edges).

## 주의사항
- 플러그인이 아닌 MCP 서버이므로 Claude Code 재시작이 필요하다.
- `.codegraph/` 디렉터리는 gitignore 처리되어 레포에 포함되지 않는다.
