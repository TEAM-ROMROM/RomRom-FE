## 🧪 테스트 케이스: 홈 피드 끊김 없는 순환 + 카드 재로딩(shimmer) 제거
**이슈**: #886  **담당자**: @Cassiiopeia  **플랫폼**: Android / iOS

> 사전 조건: 내 물건 1개 이상 등록(블러 해제 상태), 추천 피드에 물품이 여러 개 노출되는 계정.

### ✅ 1. 끝 도달 시 순환 (또돌이표)
- [ ] TC-01 홈 피드를 끝까지 세로 스크롤해도 멈추지 않고 계속 내려간다
- [ ] TC-02 전체 물품을 다 본 뒤에도 처음 물품들이 이어서 다시 나타난다
- [ ] TC-03 순환 지점에서 스크롤이 끊기거나 "더 이상 없음" 상태로 막히지 않는다
- [ ] TC-04 여러 바퀴(2~3회) 연속 순환해도 계속 정상적으로 이어진다

### ✅ 2. 카드 재로딩(shimmer) 제거
- [ ] TC-05 카드를 한 칸 넘길 때 이미 본 카드에 shimmer 로딩 애니메이션이 다시 뜨지 않는다
- [ ] TC-06 카드를 위로 여러 개 넘긴 뒤 다시 아래로 스크롤하면 본 카드가 shimmer 없이 즉시 이미지 표시된다
- [ ] TC-07 순환으로 같은 물품이 다시 등장할 때도 이미지가 즉시 표시된다 (캐시 적중)
- [ ] TC-08 카드 내 이미지가 여러 장인 경우, 카드 전환 후에도 이미지 가로 스와이프/인디케이터가 정상 동작한다

### ✅ 3. 회귀 — 광고 슬롯
- [ ] TC-09 광고가 기존 패턴([아이템 2개 + 광고 1개])대로 끼어 노출된다
- [ ] TC-10 순환 후에도 광고 삽입 위치 계산이 어긋나지 않는다
- [ ] TC-11 광고 슬롯에서는 상단 알림/리포트 버튼이 숨겨진다

### ✅ 4. 회귀 — AI 추천 / 상단 버튼
- [ ] TC-12 AI 추천 버튼 탭 → 추천 카드 하이라이트가 정상 표시된다
- [ ] TC-13 카드 전환 시 AI 하이라이트가 초기화된다 (이전 동작 유지)
- [ ] TC-14 상단 알림 아이콘(미확인 뱃지 포함)과 리포트 버튼이 일반 카드에서 정상 표시·동작한다

### ⚠️ 5. 엣지 케이스
- [ ] TC-15 추천 피드 물품이 **0개**(또는 폴백 정렬 전부 0개)이면 순환하지 않고 빈 상태로 유지된다 (무한 루프 없음)
- [ ] TC-16 물품이 1~2개로 매우 적을 때 순환해도 정상 동작하고 멈추지 않는다
- [ ] TC-17 추가 로드 중 빠르게 연속 스크롤해도 중복 로드/크래시가 없다
- [ ] TC-18 네트워크 지연 상황에서 끝 도달 → 순환 로드가 늦게 와도 최종적으로 이어붙는다
- [ ] TC-19 블러 상태(내 물건 0개)에서는 스와이프가 비활성화되어 순환이 트리거되지 않는다

### 📱 6. 플랫폼별 확인
- [ ] TC-20 Android — 핵심 시나리오(TC-01, 05, 09, 12) 정상 동작
- [ ] TC-21 iOS — 핵심 시나리오(TC-01, 05, 09, 12) 정상 동작
- [ ] TC-22 iPad/대형 기기에서 순환·카드 전환 시 레이아웃 깨짐/overflow 없음
