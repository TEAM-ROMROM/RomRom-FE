### ✅ 1. 앱 복귀 트리거 — 기본 동작 (iOS/Android 공통)

- [ ] 앱 최초 설치 후 1회 실행 → 홈 버튼으로 백그라운드 전환 후 복귀 → 팝업 **미노출** (실행 2회 미만)
- [ ] 앱 2회 이상 실행 후 → 백그라운드 전환 후 복귀 → Step 1 팝업 노출 확인
- [ ] Step 1 팝업: "RomRom을 잘 이용하고 계신가요?" 문구 확인
- [ ] Step 1 [좋아요!] 탭 → Step 2 팝업 노출 확인
- [ ] Step 2 팝업: "별점 한 줄이 큰 힘이 됩니다! ⭐ 스토어에 리뷰를 남겨주실래요?" 문구 확인
- [ ] Step 2 [리뷰 남기기] 탭 → iOS 네이티브 리뷰 다이얼로그 또는 App Store 링크 열림 확인
- [ ] Step 2 [리뷰 남기기] 탭 → Android 네이티브 리뷰 다이얼로그 또는 Play Store 링크 열림 확인

---

### ✅ 2. 거래 완료 트리거 — 기본 동작

- [ ] 거래 완료 후 리뷰 작성 화면에서 완료 → 앱 사용 3일 이상 + 거래 1회 이상인 경우 Step 1 팝업 노출
- [ ] 팝업 노출 후 정상적으로 MainScreen으로 이동 확인
- [ ] 앱 사용 3일 미만인 경우 거래 완료 후 팝업 **미노출**

---

### ⚠️ 3. 쿨타임 및 재노출

- [ ] Step 1 [별로예요] 탭 → 팝업 닫힘, 14일 이내 복귀 시 팝업 **미노출**
- [ ] Step 2 [나중에] 탭 → 팝업 닫힘, 14일 이내 복귀 시 팝업 **미노출**
- [ ] [리뷰 남기기] 탭 후 → 이후 어떤 트리거에서도 팝업 **영구 미노출**

---

### ⚠️ 4. 최대 노출 횟수 제한

- [ ] 팝업 3회 노출 후 → 4번째 트리거 조건 충족 시 팝업 **미노출** (영구 비활성)
- [ ] 앱 재설치 후 → SharedPreferences 초기화되어 카운터 리셋 확인

---

### ⚠️ 5. 엣지 케이스

- [ ] 팝업이 뜬 상태에서 앱 강제 종료 후 재실행 → 쿨타임 카운트 정상 적용 확인
- [ ] Step 1 팝업 외부 탭 (barrierDismissible: false) → 팝업 닫히지 않음 확인
- [ ] Step 1 노출 → Step 2 노출 중 앱 백그라운드 전환 → 복귀 후 팝업 상태 정상 확인
- [ ] 거래 완료 트리거 + 복귀 트리거 동시 조건 충족 → 거래 완료 직후 팝업 1회만 노출 (복귀 시 쿨타임으로 미노출)
