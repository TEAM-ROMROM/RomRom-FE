### TC-01: 이메일 중복 가입 (409 / EMAIL_ALREADY_REGISTERED) 안내
| 항목 | 내용 |
|------|------|
| 전제조건 | 동일 이메일이 다른 소셜 플랫폼으로 이미 가입된 상태 |
| 절차 | 카카오 로그인 진행 → firebase-token 응답이 409 또는 code=EMAIL_ALREADY_REGISTERED → 예외 전파 |
| 기대 | `EmailAlreadyRegisteredException`이 발생하고, 로그인 화면에서 "이미 {플랫폼} 계정으로 가입된 이메일입니다" 안내가 표시됨 |

### TC-02: 등록 플랫폼명 한글 매핑 확인
| 항목 | 내용 |
|------|------|
| 전제조건 | 응답 registeredSocialPlatform가 KAKAO/GOOGLE/APPLE 중 하나 |
| 절차 | 중복 가입 케이스 재현 → 안내 메시지 확인 |
| 기대 | KAKAO→카카오, GOOGLE→구글, APPLE→Apple로 표시. 알 수 없는 값은 "다른 소셜"로 표시 |
