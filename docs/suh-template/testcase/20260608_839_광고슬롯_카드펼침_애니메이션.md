### TC-04: 인앱 리뷰 카드 표출 시 스토어 폴백 안 함

| 항목 | 내용 |
|------|------|
| 전제조건 | 인앱 리뷰 가능 환경(`isAvailable=true`) |
| 절차 | 리뷰 트리거 조건 충족 후 [리뷰 남기기] 클릭 |

### TC-05: 인앱 리뷰 카드 미표출 시 스토어 폴백

| 항목 | 내용 |
|------|------|
| 전제조건 | 인앱 리뷰 카드가 뜨지 않는 상황(쿼터 소진 등) |
