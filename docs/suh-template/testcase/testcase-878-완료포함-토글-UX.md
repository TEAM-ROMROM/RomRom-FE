## 🧪 테스트 케이스: 요청관리 완료 포함 토글 UX 개선
**이슈**: #878  **담당자**: @Cassiiopeia

### ✅ 1. 기본 기능 동작 확인

#### 받은 요청 탭
- [ ] 토글 OFF(기본) 상태에서 진행 중인 요청만 표시되는지 확인
- [ ] 토글 ON 시 진행 중 + 완료된 요청이 함께 표시되는지 확인
- [ ] 토글 옆 라벨이 '완료 포함'으로 표시되는지 확인
- [ ] 토글 ON/OFF 전환 시 목록이 즉시 갱신되는지 확인

#### 보낸 요청 탭
- [ ] 보낸 요청 탭에서도 동일한 토글이 동작하는지 확인
- [ ] 받은 요청 탭에서 토글 변경 시 보낸 요청 탭에도 상태가 동기화되는지 확인

### ⚠️ 2. 엣지 케이스 테스트

- [ ] 완료된 요청이 없을 때 토글 ON → '진행 중인 요청이 없습니다' 메시지 표시 확인
- [ ] 진행 중 요청도 없고 완료 요청도 없을 때 → 빈 상태 메시지 정상 표시 확인
- [ ] 완료 요청만 있고 진행 중 요청이 없을 때 토글 ON → 완료 항목만 표시되는지 확인
- [ ] 토글 ON 상태에서 탭 전환(받은↔보낸) 후 돌아올 때 토글 상태 유지 확인

### 📱 3. 플랫폼별 확인

- [ ] Android에서 토글 애니메이션 정상 동작 확인
- [ ] iOS에서 토글 애니메이션 정상 동작 확인
- [ ] iPad(태블릿)에서 레이아웃 깨짐 없는지 확인

### 🎨 4. UI/UX 확인

- [ ] 토글 라벨 '완료 포함' 텍스트 잘림 없는지 확인
- [ ] 완료된 항목이 진행 중 항목과 시각적으로 구분되는지 확인 (거래 상태 표시)
- [ ] 스크롤 시 정렬 칩 + 토글 영역 레이아웃 유지 확인
