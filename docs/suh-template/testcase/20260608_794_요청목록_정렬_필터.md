### TC-01: 받은 요청 최신순 정렬(기본)

| 항목 | 내용 |
|------|------|
| 전제조건 | 받은 교환 요청이 여러 건 존재 |
| 절차 | 요청 관리 → 받은 요청 탭 진입 |
| 기대 | 기본 정렬 `최신순`(`sortField=CREATED_DATE`, `sortDirection=DESC`)으로 최근 요청이 위에 표시 |

### TC-02: 가격 높은순 / 낮은순 정렬

| 항목 | 내용 |
|------|------|
| 절차 | 정렬 칩에서 `가격 높은순` 선택 → 다시 `가격 낮은순` 선택 |
| 기대 | 높은순=`PRICE/DESC`, 낮은순=`PRICE/ASC`로 서버 재조회되어 가격 기준 재정렬 |
