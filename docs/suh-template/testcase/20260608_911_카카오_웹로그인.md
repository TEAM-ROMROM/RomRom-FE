### TC-03: 백엔드 Custom Token으로 Firebase 로그인 (UID 고정 확인)
| 항목 | 내용 |
|------|------|
| 전제조건 | 카카오 accessToken 발급 완료 |
| 절차 | 카카오 로그인 → `/api/auth/kakao/firebase-token` 요청 → 응답의 customToken으로 Firebase signInWithCustomToken 수행 |
| 기대 | 동일 카카오 계정 재로그인 시 Firebase UID가 `kakao:{회원번호}` 형태로 항상 동일하게 유지됨 (OIDC credential 기반 변동 없음) |
| 결과 | |
