### TC-02: 재연결 동기화 중복 실행 방지
| 항목 | 내용 |
|------|------|
| 전제조건 | 채팅방 진입 상태, 네트워크 불안정으로 짧은 시간 내 재연결 이벤트 다발 발생 |
| 절차 | 네트워크를 빠르게 여러 번 토글하여 onReconnected 이벤트가 연속 발생하도록 유도 |

### TC-03: 내 텍스트 메시지 낙관적 즉시 표시
| 항목 | 내용 |
|------|------|
| 전제조건 | 채팅방 진입 상태 |
| 절차 | 텍스트 입력 후 전송 버튼 클릭 |
| 기대 | WS 에코를 기다리지 않고 내 메시지가 즉시 목록 최상단에 표시되고 자동 스크롤됨. 에코 도착 시 실제 서버 메시지로 교체되며 중복 표시되지 않음 |

### TC-05: 메시지 표시 시각 정확성 (시각 폴백 제거 검증)
| 항목 | 내용 |
|------|------|
| 전제조건 | 채팅방 진입, 메시지 지연 수신 상황 유도 |
| 절차 | 단절 후 지연 수신된 메시지의 표시 시각 확인 → 재연결 동기화 후 시각 재확인 |
| 기대 | DateTime.now() 폴백이 제거되어 지연 수신 메시지가 '수신 시각'으로 밀리지 않음. 시각 출처 없으면 재동기화 시 서버 createdDate로 교정됨 |
| 결과 | |
