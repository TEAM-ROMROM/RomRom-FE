🗒️ 설명
---

카테고리 선택 화면에서 제거 예정인 카테고리 4종(`키덜트`, `주얼리`, `시계`, `식품`)을 목록에서 삭제해야 한다.

현재 `lib/enums/item_categories.dart`에 정의된 카테고리 목록:
- `watches` (id: 5, label: '시계', serverName: 'WATCHES') → **제거**
- `jewelry` (id: 6, label: '주얼리', serverName: 'JEWELRY') → **제거**
- `kidult` (id: 14, label: '키덜트', serverName: 'KIDULT') → **제거**
- `food` (id: 22, label: '식품', serverName: 'FOOD') → **제거**

또한 현재 카테고리 선택 화면(`재능/서비스`, `기타`)의 아이콘이 비어있는 상태로 노출되고 있어 UI 개선이 함께 필요하다.

🛠️ 변경 범위
---

- `lib/enums/item_categories.dart` — 4개 enum 값 제거
- `lib/screens/onboarding/category_selection_step.dart` — 카테고리 선택 화면 반영
- `lib/screens/my_page/my_category_settings_screen.dart` — 마이페이지 카테고리 설정 화면 반영
- 해당 카테고리를 참조하는 곳 전수 확인 필요
