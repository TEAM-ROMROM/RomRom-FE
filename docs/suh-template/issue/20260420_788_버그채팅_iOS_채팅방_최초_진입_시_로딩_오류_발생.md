🗒️ 설명
---

iOS 환경에서 채팅방에 처음 진입할 때 오류가 발생한다. 재시도 버튼을 누르면 정상적으로 동작한다.

- `ChatRoomScreen` 진입 시 `_loadInitialData()`에서 WebSocket 연결(`_wsService.connect()`) 또는 채팅 메시지 API 호출(`ChatApi().getChatMessages()`) 단계에서 간헐적으로 실패하는 것으로 추정
- 오류 발생 시 `_hasError = true`로 전환되어 에러 화면이 표시됨
- 재시도 시 WebSocket 세션이 이미 생성된 상태(`_stompClient != null`)이므로 연결 단계를 건너뛰고 API 호출만 재시도되어 성공하는 것으로 추정

🔄 재현 방법
---

1. iOS 기기에서 앱 실행
2. 채팅 탭 진입
3. 채팅방 목록에서 채팅방 선택 (최초 진입)
4. 오류 화면 표시 확인
5. 재시도 버튼 클릭 → 정상 진입 확인

📸 참고 자료
---

- 관련 파일:
  - `lib/screens/chat_room_screen.dart` — `_loadInitialData()` 초기화 로직
  - `lib/services/chat_websocket_service.dart` — `connect()` WebSocket 연결 처리

✅ 예상 동작
---

- iOS에서 채팅방 최초 진입 시 오류 없이 정상적으로 메시지 목록이 로드되어야 한다.
