🗒️ 설명
---

iOS AdMob 심사가 통과되어 프로덕션 AdMob 광고 Unit ID를 적용한 빌드를 App Store에 새로 배포해야 한다.

현재 상태:
- AdMob iOS 앱 심사 통과 완료
- 현재 빌드는 테스트 광고 Unit ID 또는 미설정 상태로 배포되어 있을 수 있음
- 프로덕션 AdMob Unit ID로 교체 후 재빌드 → App Store Connect 제출 필요

🛠️ 작업 내용
---

1. `.env` 또는 설정 파일에 프로덕션 AdMob iOS Unit ID 적용 확인
2. iOS 릴리즈 빌드 (`flutter build ipa`)
3. App Store Connect에 새 버전 업로드 (Transporter 또는 Xcode)
4. App Store Connect에서 심사 제출
5. 심사 통과 후 출시

📸 참고 자료
---

- 관련 파일:
  - `lib/services/ad_mob_service.dart` — AdMob Unit ID 로드 로직
  - `lib/widgets/native_ad_widget.dart` — 네이티브 광고 위젯

✅ 예상 동작
---

- App Store 배포 버전에서 실제 AdMob 광고가 정상 노출되어야 한다.
