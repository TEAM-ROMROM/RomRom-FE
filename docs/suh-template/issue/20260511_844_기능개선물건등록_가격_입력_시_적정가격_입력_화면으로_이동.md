## 📝 현재 문제점

- `lib/widgets/register_input_form.dart`의 가격 입력 TextField를 터치하면 인라인으로 가격을 직접 입력하는 구조
- AI 추천 가격 토글과 가격 입력 필드가 같은 화면에 혼재되어 UX가 복잡함
- 가격 입력 영역이 작아 AI 추천 가격 활성화 흐름이 직관적이지 않음

## 🛠️ 해결 방안 / 제안 기능

- 가격 입력 TextField 탭 시 **별도의 적정가격 입력 화면(BottomSheet 또는 새 Screen)으로 이동**
- 해당 화면에서:
  - AI 추천 가격 토글 버튼 제공
  - 직접 가격 입력 필드 제공
  - 확인 버튼으로 선택한 가격을 등록 폼에 반영

## ⚙️ 작업 내용

- 적정가격 입력 화면 신규 위젯/스크린 구현 (`lib/screens/` 또는 `lib/widgets/`)
- `register_input_form.dart`의 가격 TextField `onTap` 핸들러에서 해당 화면으로 이동 처리
- AI 추천 가격 토글 로직(`_measureAiPrice`, `useAiPrice`) 새 화면으로 이전
- 가격 확정 후 `priceController` 및 `useAiPrice` 상태 반영

관련 파일:

- `lib/widgets/register_input_form.dart`
