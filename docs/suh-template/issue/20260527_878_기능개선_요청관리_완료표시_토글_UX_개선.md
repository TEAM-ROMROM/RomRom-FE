# 🚀[기능개선][요청관리] 완료 표시 토글 UX 개선

- 라벨: 작업전
- 담당자: Cassiiopeia

---

📝 현재 문제점
---

- 요청 관리 화면의 "완료 표시" 토글이 ON일 때 완료된 요청만 표시되고 진행 중인 요청은 숨겨짐
- 일반적인 앱 UX 기준에서 완료 토글은 "완료 항목도 함께 보기" 의미이나, 현재는 완료 항목만 단독으로 표시됨
- 관련 파일: `lib/screens/request_management_tab_screen.dart`

🛠️ 해결 방안 / 제안 기능
---

- 토글 OFF(기본): 진행 중인 요청만 표시 (기존 동일)
- 토글 ON: 진행 중 + 완료된 요청 모두 표시 (변경)
- 완료된 항목은 시각적으로 구분 가능하도록 처리

⚙️ 작업 내용
---

- `request_management_tab_screen.dart` 내 `_showCompletedRequests` 필터링 로직 수정
- 토글 ON 시 `TRADED` 상태 포함 전체 표시로 변경
