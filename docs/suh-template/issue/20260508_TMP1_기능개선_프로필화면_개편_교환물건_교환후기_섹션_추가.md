📝 현재 문제점
---

- 현재 프로필 화면(`member_profile_screen.dart`)에는 닉네임·위치·받은 좋아요 수만 표시됨
- 해당 유저가 등록한 교환 물건 목록을 프로필 내에서 바로 확인할 수 없음
- 교환 후기(평점·키워드 태그·리뷰 댓글)가 프로필 화면에 전혀 노출되지 않아 신뢰도 파악이 어려움

🛠️ 해결 방안 / 제안 기능
---

디자인 시안 기준으로 프로필 화면을 아래와 같이 개편한다.

**1. 프로필 헤더 영역 변경**
- 닉네임 옆에 편집 아이콘(연필) 배치 (내 프로필일 때만)
- 아바타 좌하단에 카메라 아이콘 오버레이 (내 프로필일 때만)
- 위치·받은 좋아요 수는 카드 형태로 유지

**2. 교환 물건 섹션 추가 (신규)**
- 섹션 제목: `교환 물건` + 우측 `>` 이동 버튼
- 가로 스크롤 아이템 카드 목록
- 각 카드: 이미지 썸네일, 상태 뱃지(채팅중 / 교환완료), 제목, 가격, 지역
- 관련 파일: `lib/screens/profile/member_profile_screen.dart`, `lib/widgets/profile_sections.dart`

**3. 교환 후기 섹션 추가 (신규)**
- 섹션 제목: `교환 후기`
- 세 가지 평점 이모지 + 카운트 표시 (별로예요 / 좋아요 / 최고예요)
- 키워드 태그 칩 목록 (사진과 같아요, 약속을 잘 지켜요, 친절해요, 물건 상태가 좋아요, 답장이 빨라요) + 각 태그별 카운트
- 후기 댓글 목록 (리뷰어 아바타, 닉네임, 지역, 내용 미리보기 + `∨` 펼치기)
- 관련 파일: `lib/screens/profile/member_profile_screen.dart`, `lib/widgets/profile_sections.dart`
