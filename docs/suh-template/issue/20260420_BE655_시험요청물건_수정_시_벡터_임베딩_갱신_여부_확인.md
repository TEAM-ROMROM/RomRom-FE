🗒️ 설명
---

물건 수정(업데이트) API 호출 시 해당 물건의 벡터 임베딩 값이 정상적으로 갱신되는지 확인이 필요하다.

- 물건 등록 시 제목/설명/카테고리 등 텍스트 기반으로 벡터 임베딩이 생성됨
- 물건 수정 시 변경된 내용이 벡터에 반영되지 않으면 AI 추천/매칭 정확도에 영향을 줄 수 있음
- 현재 물건 수정 API에서 벡터 재계산 로직이 포함되어 있는지 확인 필요

🔄 확인 방법
---

1. 물건 등록 후 벡터 값 확인
2. 물건 제목/설명/카테고리 수정
3. 수정 후 벡터 값이 변경되었는지 확인
4. 변경되지 않았다면 물건 수정 API에 벡터 재계산 로직 추가 필요

✅ 예상 동작
---

- 물건 수정 시 변경된 내용(제목, 설명, 카테고리 등)이 벡터 임베딩에 즉시 반영되어야 한다.
- AI 추천/매칭 시 수정된 최신 벡터 값이 사용되어야 한다.
