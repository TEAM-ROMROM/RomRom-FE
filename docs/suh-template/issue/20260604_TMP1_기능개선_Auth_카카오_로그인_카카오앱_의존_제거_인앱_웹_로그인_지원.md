📝 현재 문제점
---

- 현재 카카오 로그인은 기기에 **카카오톡 앱이 설치되어 있어야만** 동작함
- Apple App Store 심사에서 **Guideline 4.2.3(i) - Design - Minimum Functionality** 위반으로 반려
  - Submission ID: `4a91d971-58ea-4327-a113-a4a2f827285d`
  - 심사일: 2026-06-04 / 기기: iPhone 17 Pro Max / 버전: 1.10.5 (163)
- 사용자가 앱 내에서 카카오 로그인을 완료하려면 별도 앱(카카오톡)을 먼저 설치해야 하는 구조
- 관련 파일: `lib/services/kakao_auth_service.dart`

🛠️ 해결 방안 / 제안 기능
---

- 카카오톡 앱 미설치 환경에서도 로그인 가능하도록 **인앱 웹 로그인 폴백** 추가
- Apple 권고사항: Safari View Controller(Flutter: `flutter_inappwebview` 또는 `webview_flutter`) 를 사용해 카카오 계정 웹 로그인 제공
- 카카오 SDK의 `isKakaoTalkInstalled()` 감지 → 미설치 시 `loginWithKakaoAccount()`(웹 로그인)로 자동 전환

⚙️ 작업 내용
---

- `lib/services/kakao_auth_service.dart` — 카카오톡 앱 설치 여부 분기 처리 추가 (앱 설치 시 카카오톡 로그인, 미설치 시 웹 로그인)
- 카카오 SDK `UserApi.instance.loginWithKakaoTalk` → `UserApi.instance.loginWithKakaoAccount` 폴백 로직
