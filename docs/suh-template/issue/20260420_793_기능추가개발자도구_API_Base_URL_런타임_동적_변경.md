🗒️ 설명
---

개발자 도구에서 API Base URL을 런타임에 동적으로 변경할 수 있는 기능이 필요하다.

현재 상황:
- `lib/models/app_urls.dart`의 `AppUrls.baseUrl`이 `const`로 하드코딩되어 있어 런타임 변경 불가
- 백엔드 PR Preview URL(`http://romrom-pr-582.pr.suhsaechan.kr:8079` 등)로 전환하려면 코드 수정 후 재빌드 필요
- 프론트 개발자가 백엔드 PR 서버를 대상으로 테스트할 때마다 재빌드가 필요해 비효율적

원하는 동작:
- 개발자 도구 플로팅 버튼 메뉴에 **"서버 URL 변경"** 항목 추가
- 패널에서 현재 Base URL 표시 + 직접 입력 또는 프리셋(prod / PR preview) 선택 가능
- 변경 즉시 이후 모든 API 호출에 새 Base URL 적용
- 앱 재시작 시 기본값(prod)으로 복원 or 선택적으로 유지

🛠️ 구현 범위
---

- `lib/models/app_urls.dart` — `baseUrl`을 `const` → 런타임 변경 가능한 방식으로 전환 (예: `static String get baseUrl`)
- `lib/debug/widgets/debug_overlay_manager.dart` — 메뉴에 "서버 URL 변경" 항목 추가
- `lib/debug/widgets/` — URL 변경 패널 위젯 신규 추가 (현재 URL 표시, 텍스트 입력, 프리셋 버튼)
- `lib/services/api_client.dart` — `const` URL 문자열 인터폴레이션 → 동적 getter로 교체

📸 참고 자료
---

- 관련 파일:
  - `lib/models/app_urls.dart:3` — `static const String baseUrl = "https://api.romrom.suhsaechan.kr"`
  - `lib/debug/debug_overlay_manager.dart:72~78` — 디버그 메뉴 항목 정의
  - `lib/services/api_client.dart:409` — `const String url = '${AppUrls.baseUrl}/...'` 패턴 다수
- Prod URL: `https://api.romrom.suhsaechan.kr`
- PR Preview URL 예시: `http://romrom-pr-582.pr.suhsaechan.kr:8079`

✅ 예상 동작
---

1. 개발자 도구 플로팅 버튼 → "서버 URL 변경" 선택
2. 현재 URL 표시 + 입력창 또는 프리셋 목록 제공
3. URL 변경 후 이후 API 호출부터 새 URL 적용
4. 현재 연결 중인 URL이 패널 어딘가에 항상 표시됨
