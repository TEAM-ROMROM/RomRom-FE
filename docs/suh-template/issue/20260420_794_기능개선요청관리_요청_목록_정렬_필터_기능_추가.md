🗒️ 설명
---

요청 관리 탭의 요청 목록에 정렬/필터 기능이 없어 사용자가 원하는 기준으로 요청을 탐색하기 어렵다.

현재 상태:
- 받은 요청 목록: 완료 여부 토글 필터만 존재 (`lib/screens/request_management_tab_screen.dart:663`)
- 보낸 요청 목록: 필터링 없이 전체 요청 표시 (`lib/screens/request_management_tab_screen.dart:290`)
- 정렬 기준 선택 UI 없음

🛠️ 명세 필요 항목
---

정렬/필터 옵션 후보 (기획 확정 필요):

**정렬 기준**
- 최신순 (기본값)
- 가격 높은순
- 가격 낮은순
- 가격 비슷한순 (내 물건 가격 기준)

**필터 기준**
- 추천순 (AI 기반 매칭 점수 순)
- 내 선호 카테고리 우선

> 위 항목 중 실제 구현할 옵션은 백엔드 API 명세와 함께 확정 필요

📸 참고 자료
---

- 관련 파일:
  - `lib/screens/request_management_tab_screen.dart:290` — 보낸 요청 목록 (정렬 없음)
  - `lib/screens/request_management_tab_screen.dart:541~663` — 받은 요청 목록 헤더 및 필터 토글
  - `lib/screens/request_management_tab_screen.dart:705~736` — 완료 여부 필터링 로직

✅ 예상 동작
---

- 요청 목록 상단에 정렬/필터 선택 UI 제공
- 선택한 기준에 따라 목록이 실시간 재정렬됨
- 받은 요청 / 보낸 요청 탭 각각 독립적인 정렬 상태 유지
